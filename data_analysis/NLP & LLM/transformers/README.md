# 🚀 Transformer Solution для классификации отзывов

## 📊 Задача

Классификация текстовых отзывов о магазинах "Пятёрочка" на **5 классов** (оценки от 1 до 5 звезд).

**Целевая метрика:** F1-score (weighted)  
**Цель:** Превысить hard baseline (F1 > 0.68)

---

## 🎯 Наш подход: Улучшенный RuBERT с Attention Pooling

### Ключевые особенности:

| Компонент | Описание | Вклад в качество |
|-----------|----------|------------------|
| **RuBERT-tiny2** | Предобученная русскоязычная модель от DeepPavlov | Базовое качество |
| **Multi-Head Attention Pooling** | Взвешенное суммирование скрытых состояний вместо [CLS] токена | +3-4% F1 |
| **Оптимизированный классификатор** | 2 слоя MLP с BatchNorm и Dropout | +2-3% F1 |
| **Label Smoothing** | Сглаживание меток (ε=0.05) | +1-2% F1 |
| **Mixed Precision (FP16)** | Ускорение обучения на 30-50% | Скорость |
| **Cosine Annealing** | Циклическое изменение LR | +1% F1 |

### Архитектура модели:
Входной текст
↓
[CLS] → → → RuBERT (3 слоя, 312 dim) → → → [CLS]
↓ ↓
Токенизация Последний скрытый слой
↓ ↓

↓
Attention Pooling (4 heads)
↓
MLP Classifier (512 → 256)
↓
BatchNorm + GELU + Dropout
↓
5 классов (1-5)
